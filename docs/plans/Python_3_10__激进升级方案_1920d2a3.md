## 用户需求

用户选择**方案 B（激进升级）**，要求将 FilterFusion 项目从 Python 3.8+ 升级到 Python 3.10+，包括：

1. 更新 Python 版本要求到 3.10+
2. 更新 `requests` 依赖版本（放宽版本约束）
3. 为代码添加完整的类型注解（修复 104 个 basedpyright 警告）
4. 创建现代 Python 项目配置（`pyproject.toml`）
5. 修复之前被回退的代码修改（`merge_rules.py` 第 317 行拼写错误）
6. 更新所有相关文档

## 功能内容

- 修改 `requirements.txt` 依赖版本
- 为 `scripts/fetch_rules.py` 和 `scripts/merge_rules.py` 添加类型注解
- 创建 `pyproject.toml` 统一管理项目配置
- 创建 `requirements-dev.txt` 添加开发依赖
- 更新 `PROJECT_DOCS.md`、`README.md`、`README_EN.md` 中的 Python 版本要求
- 修复 `merge_rules.py` 中被回退的修改

## 技术栈

- **编程语言**: Python 3.10+
- **依赖管理**: `requirements.txt` + `pyproject.toml`
- **类型检查**: basedpyright, mypy
- **测试框架**: pytest

## 实施方案

### 1. 更新依赖版本

**文件**: `requirements.txt`

```txt
requests>=2.31.0
```

**原因**: `requests` 3.0 不存在，`<3.0.0` 约束不必要。

### 2. 添加类型注解到 `scripts/fetch_rules.py`

**修改内容**:

- 添加导入: `from typing import Any, Optional`
- 为 `RuleFetcher` 类属性添加类型注解:
- `project_root: Path`
- `rules_dir: Path`
- `meta_file: Path`
- `session: requests.Session`
- 为方法添加类型注解:
- `load_sources() -> list[dict[str, Any]]`
- `fetch_single_rule(source: dict[str, Any]) -> Optional[dict[str, Any]]`
- `fetch_all_rules() -> dict[str, Any]`
- 修复第 7 行导入拼写错误: `concurrent.futures` (当前是 `futures`)

### 3. 添加类型注解到 `scripts/merge_rules.py`

**修改内容**:

- 添加导入: `from typing import Any, Optional, Union`
- 为 `RuleMerger` 类属性添加类型注解:
- `project_root: Path`
- `dist_dir: Path`
- `rules_dir: Path`
- `initial_rule_count: int`
- `final_rule_count: int`
- `start_time: datetime`
- 为方法添加类型注解:
- `load_metadata() -> dict[str, Any]`
- `load_header_template() -> str`
- `generate_source_list(sources: list[dict[str, Any]]) -> str`
- `calculate_source_stats(sources: list[dict[str, Any]]) -> tuple[list[dict[str, Any]], defaultdict[str, int]]`
- `process_rule(line: str) -> tuple[Optional[str], Optional[str]]`
- `collect_and_process_rules(sources: list[dict[str, Any]], rule_counts: defaultdict[str, int]) -> tuple[list[str], dict[str, set[str]]]`
- `generate_version() -> str`
- `merge() -> None`
- `save_summary(version: str, checksum: str, source_stats: list[dict[str, Any]], processing_time: float) -> None`
- **修复第 317 行拼写错误**: `self.final_rule_count` (当前是 `self.final_rule_count`)

### 4. 创建 `pyproject.toml`

**内容**:

- 项目元数据 (name, version, description, authors, license)
- Python 版本要求: `requires-python = ">=3.10"`
- 依赖配置: 从 `requirements.txt` 导入
- 开发依赖: `[project.optional-dependencies] dev`
- 工具配置: `[tool.mypy]`, `[tool.pytest]`, `[tool.basedpyright]`

### 5. 创建 `requirements-dev.txt`

**内容**:

```
mypy>=1.0.0
pytest>=7.0.0
basedpyright>=1.0.0
types-requests>=2.31.0
black>=23.0.0
isort>=5.0.0
```

### 6. 更新文档

**文件**:

- `PROJECT_DOCS.md` 第 110 行: `Python 3.8+` → `Python 3.10+`
- `README.md` 第 110 行: `Python: 3.8 或更高版本` → `Python: 3.10 或更高版本`
- `README_EN.md` 对应位置

### 7. 配置类型检查工具

**创建文件**: `pyproject.toml` 添加:

```
[tool.mypy]
python_version = "3.10"
strict = true

[tool.basedpyright]
pythonVersion = "3.10"
typeCheckingMode = "strict"
```
