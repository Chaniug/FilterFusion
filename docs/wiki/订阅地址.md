# 订阅地址

本文档汇总所有平台的规则订阅地址，以及导入方法。

## 规则文件地址

| 规则 | 文件名 | 订阅地址 |
|------|--------|----------|
| 手机端 AdBlock | `adblock-mo.txt` | `https://chaniug.github.io/FilterFusion/adblock-mo.txt` |
| 电脑端 AdBlock | `adblock-pc.txt` | `https://chaniug.github.io/FilterFusion/adblock-pc.txt` |
| DNS 过滤 | `dns-blocklist.txt` | `https://chaniug.github.io/FilterFusion/dns-blocklist.txt` |

> 如果配置了自定义组合规则，订阅地址格式为：
> `https://chaniug.github.io/FilterFusion/<你的文件名>.txt`

## 各工具导入方法

### uBlock Origin（推荐）

1. 打开 uBlock Origin 设置
2. 进入「过滤器列表」→「自定义」
3. 点击「从 URL 导入」
4. 粘贴对应的订阅地址
5. 点击「应用更改」

**订阅地址**：
```
https://chaniug.github.io/FilterFusion/adblock-mo.txt
https://chaniug.github.io/FilterFusion/adblock-pc.txt
```

### AdGuard（桌面端 / 浏览器扩展）

1. 打开 AdGuard 设置
2. 进入「过滤器」→「自定义过滤器」
3. 点击「新建过滤器」
4. 填写名称（如 `FilterFusion`）和订阅地址
5. 点击「保存」

**订阅地址**：
```
https://chaniug.github.io/FilterFusion/adblock-mo.txt
https://chaniug.github.io/FilterFusion/adblock-pc.txt
```

### AdGuard for Android / iOS

1. 打开 AdGuard 应用
2. 进入「过滤器」→「自定义过滤器」
3. 点击「新建过滤器」
4. 填写名称和订阅地址
5. 启用该过滤器

**手机端订阅地址**：
```
https://chaniug.github.io/FilterFusion/adblock-mo.txt
```

### AdGuard Home（DNS 过滤）

1. 打开 AdGuard Home 管理页面
2. 进入「过滤器」→「DNS 过滤」
3. 点击「添加黑名单」
4. 选择「从 URL 添加」
5. 填写名称和订阅地址

**DNS 订阅地址**：
```
https://chaniug.github.io/FilterFusion/dns-blocklist.txt
```

### Pi-hole（DNS 过滤）

1. 打开 Pi-hole 管理页面
2. 进入「Group Management」→「Adlists」
3. 在「Address」填入订阅地址
4. 点击「Add」

**DNS 订阅地址**：
```
https://chaniug.github.io/FilterFusion/dns-blocklist.txt
```

## 自动更新机制

所有规则文件每天通过 GitHub Actions 自动更新：

- **抓取** — 每天自动从源地址下载最新规则
- **合并** — 自动去重、分类、生成最终文件
- **推送** — 自动提交到仓库，GitHub Pages 自动部署

你无需手动更新，广告拦截工具会自动拉取最新版本。

## 规则文件格式

### AdBlock 规则（`adblock-*.txt`）

- 格式：ABP 标准（Adblock Plus 2.0）
- 兼容：uBlock Origin、AdGuard、Adblock Plus、Brave Browser
- 头部：含 Title、Description、Version、Checksum 等元数据
- Checksum：ABP 标准 MD5 + Base64（24 字符）

### DNS 规则（`dns-blocklist.txt`）

- 格式：AdGuard DNS 过滤格式
- 兼容：AdGuard Home、Pi-hole、DNSCrypt、Clash Premium
- 语法：`||domain^`（拦截域名）、`@@||domain^`（放行域名）
