# 安装与快速开始

## 系统要求

| 项目 | 要求 |
|------|------|
| 🐍 Python | 3.14+ |
| 💻 操作系统 | Windows / macOS / Linux |
| 🌐 网络 | 需要互联网连接以抓取规则源 |
| 📦 依赖 | `httpx[http2]>=0.27.0`、`pyyaml>=6.0`（仅此两个） |

## 第一步：克隆项目

```bash
git clone https://github.com/Chaniug/FilterFusion.git
cd FilterFusion
```

## 第二步：安装依赖

```bash
pip install -r requirements.txt
```

> 推荐使用 [`uv`](https://github.com/astral-sh/uv) 加速安装：
> ```bash
> uv pip install -r requirements.txt
> ```

## 第三步：抓取并合并规则

### 方式 A：统一入口（推荐）

```bash
# 同时抓取 AdBlock + DNS 规则
python -m scripts.fetch_all

# 合并所有规则
python -m scripts.merge_all
```

### 方式 B：分步执行

| 管道 | 抓取 | 合并去重 |
|------|------|---------|
| AdBlock (Mobile) | `python scripts/fetch_rules.py` | `python scripts/merge_rules.py --category mobile` |
| AdBlock (PC) | `python scripts/fetch_rules.py` | `python scripts/merge_rules.py --category pc` |
| DNS | `python scripts/fetch_dns_rules.py` | `python scripts/merge_dns_rules.py` |

## 第四步：查看生成的规则

生成的规则文件位于 `dist/` 目录：

```
dist/
├── adblock-mo.txt      # 手机端规则
├── adblock-pc.txt      # 电脑端规则
└── dns-blocklist.txt   # DNS 过滤规则
```

## 使用 GitHub Actions 自动更新

Fork 本项目后，GitHub Actions 会自动：

- **每日更新**（`daily-update.yml`）— 每天自动抓取、合并、提交
- **每周发布**（`weekly-release.yml`）— 每周创建 Release Tag

无需任何手动操作，规则每天自动保持最新。
