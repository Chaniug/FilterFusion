## 项目特色

- 📦 **多源聚合** — 7+ 个规则源，按类别自动分发
- 🔄 **每日自动更新** — GitHub Actions 每天自动抓取合并
- ⚡ **高效去重** — 合并时自动去重，规则数精简
- 📱 **手机/电脑分离** — 移动端和桌面端规则独立输出
- 🛡️ **DNS 规则** — 同时支持 AdBlock 和 DNS 过滤规则
- 🔧 **配置驱动** — 新增/修改规则只需编辑 YAML，无需改代码
